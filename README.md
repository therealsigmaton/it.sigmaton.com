# it.sigmaton.com (Static GitHub Pages)

This is a **static** site for GitHub Pages that reuses your Shopify theme **assets**.
Shopify Liquid templates do not run on GitHub Pages, so this package includes:
- index.html + /services /about /contact pages
- /assets copied from the Shopify theme
- minimal Sigmaton overrides (white background + brand accents)

Loaded CSS: base.css
Loaded JS: theme.js
